3D Human Avatar Creation Using Artificial Intelligence

Project Overview

This project focuses on developing a pipeline to create highly realistic 3D human avatars from a single 2D image using AI-based tools such as Character Creator 4 and NVIDIA Audio2Face. The objective was to generate real-time, animated avatars that respond to audio and text inputs, suitable for deployment in educational and telepresence applications.

I contributed by designing the workflow, creating avatars using Character Creator and Audio2Face, integrating avatars into a web application, and automating the text-to-speech interaction system.
